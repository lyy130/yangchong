1.1.1. 1.获取首页某个标签的详情数据

  • 请求地址: /user/homepage
  • 请求参数:
    'userId' int //用户id 必填
'userKey' string //用户key 必填
'tagId' int //标签id 必填  
'page' int //页码 必填
'province' string //省 如果是其他标签的时候传空字符串  第一次也传空
  • 请求参数 tagId 说明，通过登录注册接口返回:
    99999：关注
99998：推荐
99997：新人
99996：热门
99995：附近
99994：好友(绝美)
3、4、5：星级
  • 返回结果:
'result' bool //返回状态

'currPage' int//当前页码
'data' array //数据数组，格式如下:
    'id' int//用户id
    'nickname' string//用户昵称
    'topic' string//主题
    'vcoinPerMinute' int//每分钟v币
    'level' int//等级
    'online' int//在线状态 0离线 1在线 2活跃 3在聊 4勿扰 5直播recharge-package-list
    'avatar' string//头像
    'isOneFree' int//是否有限免 1:是  0:否
    'cornerMark' array // 封面角标 （6.1.4 新增）
        'id' int // 编号
        'path' string // 图片/svga地址
        'pathC' string // 图片/svga地址(C版使用)
        'isMain' int // 是否是主标 0否 1是
'isEnd' bool//是否结束 true:是  false:否
'curProvince' string //当前的省
'curImgModel' int // 当前图片应该使用的模式 1大图 0小图
